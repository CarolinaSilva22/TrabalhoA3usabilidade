Unidade Curricular 1: Análise e Projeto de Sistemas

Descrição: A unidade de Análise e Projeto de Sistemas aborda metodologias e técnicas para entender, projetar e implementar sistemas de informação que atendam às necessidades dos usuários e da organização. Envolve a análise dos requisitos do sistema e o desenvolvimento de soluções eficazes.

Objetivos de Aprendizagem: Identificar e analisar os requisitos dos sistemas de informação. Desenvolver modelos e diagramas para representar sistemas e seus processos. Aplicar metodologias de desenvolvimento de software para criar soluções eficazes. Projetar interfaces de usuário amigáveis e funcionais.

Unidade Curricular 2: Banco de Dados

Descrição: A unidade de Banco de Dados explora o design, implementação e gerenciamento de sistemas de banco de dados que armazenam e organizam dados de forma eficiente. Inclui o uso de sistemas de gerenciamento de banco de dados (SGBDs) e a aplicação de conceitos de modelagem de dados.

Objetivos de Aprendizagem: Projetar e modelar bancos de dados relacionais e não-relacionais. Utilizar SQL para gerenciar e consultar dados. Aplicar técnicas de normalização para otimizar o armazenamento e a integridade dos dados. Gerenciar e assegurar a integridade dos dados em transações.
